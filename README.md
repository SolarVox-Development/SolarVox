## Installation

> Put Your Real info in your .env
> Run this from your project folder: `npm install`
> If you‚Äôre missing core libraries, install them explicitly: `npm install discord.js dotenv`
> and for TypeScript support: `npm install --save-dev typescript ts-node @types/node`
> If this is a old token/bot that has commands run `npx ts-node scripts/reset-commands.ts`
> To deploy commands do `npx ts-node scripts/deploy-commands.ts`
> To Run the bot do `npx ts-node src/index.ts`
> If you did it right you should get `Loaded command: ping
Loaded event: messageCreate
All commands and events loaded. Logging in...
`
